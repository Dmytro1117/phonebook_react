# Phonebook

🔗 **[Live Demo](https://phonebook-alpha.vercel.app/)**

## Project Description

**Phonebook** is a comprehensive contact management application. It allows users to securely store
and organize their contact lists, create new entries, assign roles, define importance, manage
statuses, and upload photos for both user profiles and individual contacts.

## Tech Stack

- **Core:** React, Vite
- **State Management:** Redux Toolkit, Redux Persist
- **Styling:** Styled Components
- **Navigation:** React Router Dom
- **Library npm:** ANTD, Formik, Notiflix, React Loader Spinner, YUP
- **Networking:** Axios

## Getting Started

### 1. **Clone the repository:**

```bash
   git clone https://github.com/Dmytro1117/phonebook_react.git
```

### 2. **Install dependencies:**

```bash
   npm install
```

### 3. **Run the application:**

```bash
npm run start
```

The app will be available at http://localhost:5173/
